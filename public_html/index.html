About

Lay summary

Models of biological systems often include parameters which are unknown. These could be for example the reaction rate of a protein x or the initial amount of a protein y. Given the observation of a biological system with a known model but unknown parameters Bayesian inference can be used to estimate the true parameter values of the system.

In this project, Bayesian inference is performed to estimate parameters in dynamical systems which undergo a co-dimension 1 bifurcation. These are systems showing a qualitative stability change by varying exactly one parameter of the system (the bifurcation parameter). The parameters estimated in this project are the bifurcation parameter and the initial condition of the system. The estimations are performed under different conditions specified by noise, stochasticity and the number of observations.

The precision of the parameter estimation is related to the shape of the log-likelihood function. To investigate the change of the shape for different true parameter combinations in a quantitative manner, the Fisher information is calculated. Results show that favourable parameter combinations resulting in more precise parameter estimation do exist for co-dimension 1 bifurcations. This holds even when increasing noise and stochasticity or decreasing the number of observations.

Abstract

In Systems Biology parameter inference has become a well known tool to help specifying parametric mathematical models of dynamical systems. A special kind of dynamical systems are systems undergoing a co-dimension 1 bifurcation. They show a qualitative stability change when varying exactly one parameter of the system (the bifurcation parameter). In this project, first a Bayesian parameter inference approach is used to investigate the log-likelihood surface of the bifurcation parameter and the initial condition of the system of four different co-dimension 1 bifurcations. The shape is analysed for various conditions including noise, stochasticity and the number of observations. Additionally, the Fisher information around the bifurcation is estimated to gain a better understanding of the change of the shape of the log-likelihood surface. For all four bifurcations stability properties resulting in more precise parameter inference are identified.

Keywords: Bayesian inference, stability analysis, co-dimension 1 bifurcation

Background

Parameter inference in Systems Biology

In Systems Biology it is often the case that data of dynamical systems is retrieved even though there is little knowledge about the underlying theory such as the true values of the parameters of the system. The idea to determine properties of the system, given some observed behaviour is manifested in the Bayes' theorem (Formula 1),[1].

P ( A | B )   =   P ( B | A ) P ( A ) P ( B )

Formula 1: Bayes' Theorem.

The probability of a property A given the observation B is determined by the probability of the observation B given the property A times the probability of A, divided by the probability of the observation B. The probability of A is called the prior and the probability of the observation B given the property A is called the likelihood. However, in many cases the likelihood is not feasible or its calculation is computationally very expensive. One solution to overcome this issue is to use an approximation instead of the exact likelihood. In this work, an approximation via the euclidean distance proposed by Kirk et al. is used (Formula 2),[2].

log(L ( θ ; D )) ≈  -  1 2   ∑ i = 1 M d(y,μ(θ)) i 2

Formula 2: Log - Likelihood Approximation.
θ relates to property A which is a certain parameter combination of the system of interest. y is the observation of the system and μ(θ) is the simulation of the system using θ. M determines the number of observations and d represents the Euclidean distance. Intuitively the relationship of likelihood and euclidean distance is shown in Figure 1, left. The more similar a simulation of a tested parameter combination is to the true system, the higher is its likelihood to be the true value. In order to perform parameter inference, many parameter combinations are tested and an an approximated log-likelihood surface is constructed. In the case of this data analysis study, θ is a parameter vector containing the initial conditions and a parameter α of the dynamical system of interest. An example approximated log-likelihood surface is shown in Figure 1, right.

Figure 1: Left: Linear relationship of squared euclidean distance and approximated log- likelihood. Right: Log-likelihood surface of initial conditions and the bifurcation parameter α for true values(0.0,0.0).

Fisher information

The Observed Fisher information uses the log-likelihood to describe how strong the dynamical system reacts to minor changes of the parameter vector θ. The Formula is given below (Formula 5), [3].

I  (   α , init   )  =   -   δ  (   Hessian ( LL ( α  ,  init ;  D  )))

Formula 3: observed Fisher information.

The second derivative of the log-likelihood function is calculated in the Hessian. α is a parameter of the dynamical system of interest, init is the initial condition of the system and D is the observation of true system. δ is a function to reduce the dimension of the information matrix. An example for δ is the determinant or the trace. Note, that there is no expectation value calculated within the Fisher information as no noise is added in this part of the study.

stability analysis

The stability of a dynamical system can be determined by the existence and stability of fixed points. A fixed point is a state, where the system is left unchanged by dynamics (Formula 3),[4].

d(x) d(t) =   f(x,α,t)  =   0

Formula 4: Definition of a fixed point X.

A fixed point can be stable (=attracting) or unstable (=repelling). A stable fixed point is also called a sink and a unstable fixed point is called a source. The property of stability can be investigated by determining signum of the gradient of the fixed point. A negative gradient implies a stable fixed point, whereas a positive gradient implies a unstable fixed point (Formula 4). If the gradient is equal to zero, the fixed point is called marginal.

f '( x ) < 0 as t -> ∞   ⇒  X is stable             f '( x ) > 0 as t ->∞   ⇒  X is unstable

Formula 5: Stability of a Fixed point X.

Stability analysis is a very important part of this study as we want to investigate weather the existence and stability of fixed points can result in favourable conditions for parameter inference. A question we want to answer could be: Can we infer a parameter α easier if one stable fixed point exists or if two stable and one unstable fixed points exist? This question makes obviously only sense, if there are actually different quality states of the global stability of the system of interest, e.g. fixed points appear or disappear. An example for these kind of systems are systems undergoing a co-dimension 1 bifurcations.

co-dimension 1 bifurcations

Animation 1: Dynamical bifurcation diagram of the super-critical Pitchfork bifurcation.

A qualitative stability change in a dynamical system caused by a smooth change of the systems' parameters is called a bifurcation. If the change is caused by only one parameter α, the bifurcation is called a co-dimension 1 bifurcation [4],[5]. In animation 1, an example bifurcation is presented. Please press play or set the bifurcation parameter. The current fixed points labelled with their stability property will be depicted. If α is negative, one stable fixed point exists. If α is positive, two stable and one unstable fixed point exist. If α is zero one marginal fixed point exists. The animation approximates a bifurcation diagram of the example bifurcation. The complete diagram is shown in Figure 2.

Discussion

Saddle-Node bifurcation The biggest influence on the performance of parameter inference in a system with bifurcation type Saddle-Node is caused by the sigum of the bifurcation parameter α. A negative value causes an approximated log-likelihood of negative infinity as the system diverges (Example: Compare α = -0.5 and α = 0.5). Therefore, a positive α is favoured. Secondly, initial conditions below the unstable fixed point should be avoided as the approximated log-likelihood is negative infinity and the system diverges again (Example: Compare α = 0.5 with initial conditions -1 and 1). Thirdly, systems with true initial conditions between the two fixed points are prefered compared to an initial condition above the fixed points as the initial conditions can be estimated more precise (Example: Compare α = 0.5 with initial conditions -0.4 and 1). Lastly, an interesting aspect is that an initial condition closer to the unstable fixed points results in better parameter inference (Example: Compare α = 0.5 with initial conditions -0.4 and 0.4).

  • Back to likelihood analysis       

Transcritical bifurcation Again, the biggest influence on the performance of parameter inference in a system with transcritical bifurcation is caused by the sigum of the bifurcation parameter α (Example: Compare α = -0.5 and α = 0.5). Surprisingly, the shape of the log-likelihood varies significantly and positive values of α are favourable. Given a positive α, again initial condition below the unstable fixed point should be avoided as divergence would be caused (Example: Compare α = 0.5 with initial conditions -1 and 1). Lastly, differently to the saddle-node bifurcation, initial conditions above the stable fixed point allow higher precision in parameter inference (Example: Compare α = 0.5 with initial conditions 0.1 and 1.0).

  • Back to likelihood analysis       

Super-critical Pitchfork bifurcation The approximated log-likelihood is symmetrical along eth x axis and the favourable sigum of the bifurcation parameter α for the super-critical Pitchfork bifurcation is positive (Example: Compare α = -0.5 and α = 0.5). The super-critical Pitchfork bifurcation is very robust towards initial conditions. Only slight preference regarding initial conditions for α estimation is observed towards initial conditions closer to the unstable fixed point (Example: Compare α = 0.5 and initial conditions 0.1 and 0.6).

  • Back to likelihood analysis       

Sub-critical Pitchfork bifurcation The approximated log-likelihood is symmetrical along the x axis and the favourable sigum of the bifurcation parameter α for the super-critical Pitchfork bifurcation is clearly negative as positive values cause divergence. (Example: Compare α = -0.5 and α = 0.5). In contrast to the super-critical case, the sub-critical case is sensitive towards the initial conditions (Example: Compare α = -0.5 with initial conditions 0.1 and 0.6).

  • Back to likelihood analysis       

Noise, Number of observations, Stochasticity Adding a noise term results in greater distances between simulations of close candidates and the true system. Therefore, the approximated log-likelihood decreases (Example: Saddle-node bifurcation, Compare α = 0.2 and initial condition 0.4 with Noise 0 and 1). Decreasing the number of observations causes less precision. When adding heavy stochasticity, inferring parameters is a lot harder (Example: Transcritical bifurcation, Compare α = 0.5 and initial condition 0.4 with Stochasticity 0 and 1). Also it is possible that a system converges to a fixed point that it is not supposed to. Also it is observed, that a system diverges even though it should not or the other way around. This can happen in every bifurcation type while testing a parameter combination or simulating the true system (Example: Sub-critical Pitchfork bifurcation, Compare α = -0.2 and initial condition 0.6 with Stochasticity 0 and 1).

Fisher information For all four bifurcation types the fisher information is relatively high around the bifurcation at α equals zero (Example: δ = trace, log(FI) > 6.9 for all types). Also enrichment can be found around unstable fixed points. For instance, when reducing the dimension of the information matrix via the logarithm of the trace for the sub-critical Pitchfork bifurcation, we observe the fisher information increasing from 5 (init condition = 0.2) to 25 (init condition = 0.0) and decreasing again to 5 (initial condition = -0.2) for positive α. This meets the expectation as minor changes can result in either convergence or divergence. Low fisher information can be found when the true system diverges because no fixed point exists (Example: Saddle-Node bifurcation, negative α) or the system is initialized below a unstable fixed point (example: transcritical, negative initial condition, positive α).
